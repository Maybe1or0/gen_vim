================================================== INSTALLATION

1)  Clone this repository into your AFS space:

git clone

2)  Move into the project directory:

cd

3)  Run the setup script:

./setup

That’s it. GENVIM is now ready.

================================================== USAGE

Launch genvim from your terminal, with or without arguments (just like
vim):

genvim

A new virtual desktop named “genvim” will automatically be created.

To switch to it, use your system bind key (commonly the Windows key) +
w.

You are now isolated. No noise. No distractions. No excuses.
